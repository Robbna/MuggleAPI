# 🔎 Muggle API 
MuggleAPI is a JSON based API with data from the Harry Potter Universe! 🤓

## API Reference
### Base URL
```https
https://muggle-api.up.railway.app
```

#### 🙋 Get all characters
```https
GET /api/characters
```

#### 🫖 Get all potions 
```https
GET /api/potions
```

#### ✨ Get all spells 
```https
GET /api/spells
```

#### 📚 Get all books 
```https
GET /api/books
```
